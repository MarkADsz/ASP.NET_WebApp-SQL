I developed an ASP.NET Core Web Application With SQL Server Database Connection and CRUD Operations. 
I am using a local host database with Sql Express.
The user can Add a new movie or Edit/Delete an existing movie from the database displayed on the /Movies/Index page.
You can see the app in use here: https://github.com/MarkADsz/ASP.NET_WebApp-SQL/blob/506d1533e9245b978a7376527f2ad61758330dca/ShowcaseWebApp.pdf
